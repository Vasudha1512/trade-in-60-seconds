# 📈 Trade in 60 Seconds
## 📌 Overview  
**Trade in 60 Seconds** is an interactive trading simulation web application that challenges users to make quick buy or sell decisions within a limited time. The project recreates a fast-paced market environment using dynamic visuals, real-time-like charts, and an engaging dashboard UI.

## 💡 Concept  
The idea is to simulate the pressure and excitement of real-world trading in a simplified format. Users must analyze price movement and make decisions quickly before time runs out, creating a thrilling and engaging experience.

## 🎯 Features  

### 📊 Trading Dashboard  
- Clean and modern market-style UI

- Balance tracking  

### ⏱️ 60-Second Challenge  
- Countdown timer  
- Fast decision-making gameplay  

### 📉 Chart Visualization  
- Candlestick-style chart  
- Simulated price movement  
- Dynamic updates  

### 🟢 Buy / 🔴 Sell Actions  
- One-click trading actions  
- Instant feedback

### ✨ Animations  
- Smooth transitions  
- Real-time UI updates  

### 📱 Mobile-Friendly Design  
- Fully responsive layout

## 🧩 Gameplay Flow  

1. **Start Screen**  
   - User enters the trading interface  

2. **Market View**  
   - Live-style chart and price display  

3. **Decision Phase**  
   - User chooses Buy or Sell  

4. **Countdown Ends**  
   - Trade result is calculated  

5. **Result & Balance Update**  
   - Profit or loss shown
